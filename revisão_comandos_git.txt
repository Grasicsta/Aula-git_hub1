-> git init -> inicializa um repositorio local do git
-> git add <arquivos> -> adiciona na branch do repositorio local-> torna o arquivo possivel de ser vigiado/comitado´pelo git
-> git status -> passa a situação atual dos arquivos e´pastas do repositorio local
-> git commit -> ele salva o que foi desenvolvido no momento, tornando possivel de retornar aquele estado do codigo
